# SmartCoreSolution PWA

SmartCoreSolution은 헬스케어 및 스마트제조 SaaS 솔루션을 제공하는 Progressive Web App(PWA)입니다. 이 프로젝트는 AI 음성·언어 분석 기술을 활용하여 의료 및 제조 분야의 디지털 전환을 지원합니다.

## 주요 기능

- **헬스케어 서비스**
  - 정신 건강·인지 기능 진단: AI 기반 음성 분석으로 치매, 우울증 등 조기 진단 지원
  - 의료 기록 자동화: 음성 입력을 통해 EMR(전자 의료 기록)에 자동 저장
  - 재활 치료·의료 통역: 언어 재활 및 다국적 환자를 위한 실시간 의료 통역

- **스마트제조 솔루션**
  - IoT·AI·MES 기반으로 생산, 품질, 안전 데이터를 실시간 모니터링 및 공정 최적화

- **PWA 지원**
  - 오프라인 사용 가능
  - PWA 설치 지원 (Chrome/Edge/Android)
